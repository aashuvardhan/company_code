/*


Write an algorithm to determine if a number n is happy.

A happy number is a number defined by the following process:

Starting with any positive integer, replace the number by the sum of the squares of its digits.
Repeat the process until the number equals 1 (where it will stay), or it loops endlessly in a cycle which does not include 1.
Those numbers for which this process ends in 1 are happy.
Return true if n is a happy number, and false if not.

 

Example 1:

Input: n = 19
Output: true
Explanation:
12 + 92 = 82
82 + 22 = 68
62 + 82 = 100
12 + 02 + 02 = 1
Example 2:

Input: n = 2
Output: false
 

Constraints:

1 <= n <= 231 - 1

*/


 int give_next(int n){
        int sum=0;
        while(n!=0){
            int dgt=n%10;
            sum+=(dgt*dgt);
            n/=10;
        }
        return sum;
    }

    bool isHappy(int n) {
        // since there will be loop if the number is nt happy
        // so either we can keep a set of all the numbers generted using set -> take O(n) space
        // or we can use the idea of slow and fast pointers in linklist to catch the loop -> O(1) space

        int slow=n;
        int fast=give_next(n);

        while(fast!=1 and slow!=fast){
            slow=give_next(slow);
            fast=give_next(give_next(fast));
        }

        return fast==1;
    }
